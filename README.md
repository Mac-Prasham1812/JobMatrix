# JobMatrix – Android Job Portal Application

JobMatrix is a role-based Android job portal application built using Kotlin and Firebase.  
The application connects Students, Employers, and Admins in a structured job management system.

---

## 🚀 Features

### 👨‍🎓 Student Panel
- Browse available jobs
- Apply for jobs
- Track application status
- Receive notifications

### 🏢 Employer Panel
- Post new jobs
- Edit job details
- Soft delete jobs (Active/Deleted status system)
- View applications
- Shortlist / Reject candidates

### 🛠 Admin Panel
- Monitor system activities
- Manage users
- Oversee job postings

---
